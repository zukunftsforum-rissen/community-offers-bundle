# Community Offers Bundle

A Contao bundle for managing community offers.

## Installation

```bash
composer require zukunftsforum-rissen/community-offers-bundle
```

## Requirements

- PHP 8.1 or higher
- Contao 5.0 or higher
- Symfony 6.4 or 7.0

## Features

This bundle provides functionality for managing community offers in Contao CMS.

## Configuration

The bundle is automatically registered via the Contao Manager Plugin.
