
# Line-by-Line Explanation Prompt

Explain this file section by section.

For each section answer:

- What exactly does this code do?
- Why might it have been written this way?
- Which dependencies does it rely on?
- What preconditions must hold?
- What side effects occur?
- Does it match the project documentation?
- Which review questions should I ask here?

Mark especially:

- critical sections
- unclear logic
- possible error sources
- test-relevant code
